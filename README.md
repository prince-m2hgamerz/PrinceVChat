# PrinceVChat

A modern, free group voice chat app.

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

### Production

```bash
npm run build
npm start
```

---

## Deploy to Railway

1. Push to GitHub
2. Connect repo to Railway
3. Deploy automatically

## Features

- Real-time voice chat (WebRTC)
- No signup required
- Shareable invite links
- Works on all modern browsers
